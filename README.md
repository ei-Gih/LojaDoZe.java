# 🛒 Mini Sistema de Login — Loja do Zé

Projeto acadêmico desenvolvido em Java com o objetivo de praticar conceitos fundamentais de programação, como vetores, estruturas de repetição, condicionais, autenticação simples e manipulação de dados em memória.

---

## 📚 Sobre o Projeto

O sistema simula uma pequena loja virtual chamada **Loja do Zé**, permitindo que usuários:

- realizem cadastro
- façam login
- comprem produtos
- visualizem atualização de saldo

Tudo isso através de um menu interativo no terminal.

---

## 🎯 Objetivo do Exercício

Este exercício foi proposto para reforçar os seguintes conceitos:

- Vetores paralelos
- Estruturas condicionais (`if/else`)
- Estruturas de repetição (`while` e `for`)
- Manipulação de Strings
- Entrada de dados com `Scanner`
- Controle de login
- Lógica de programação

---

## ⚙️ Funcionalidades

### ✅ Cadastro de usuários
- Limite máximo de 5 usuários
- Armazenamento de:
  - nome
  - senha
  - saldo inicial

### ✅ Sistema de login
- Validação de usuário e senha
- Controle de usuário logado

### ✅ Sistema de compras
Produtos disponíveis:
- 👕 Camiseta — R$50
- 🧢 Boné — R$30

O sistema:
- verifica se existe usuário logado
- valida saldo disponível
- desconta o valor da compra
- exibe saldo atualizado

### ✅ Encerramento do sistema
- opção para sair do programa a qualquer momento

---

## 🧠 Conceitos Utilizados

```java
String[] nomes
String[] senhas
double[] saldos
```
O projeto utiliza vetores paralelos, onde cada posição dos arrays representa o mesmo usuário.
